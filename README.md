# Festival-Avatar

上传图片在线制作节日头像

![image](https://raw.githubusercontent.com/singingXY/mypic/master/makeAvatar/1.png)

点击贴纸素材添加到头像上，素材可调整大小旋转角度，添加多个素材时为方便操作可将素材锁定位置。

![image](https://raw.githubusercontent.com/singingXY/mypic/master/makeAvatar/2.png)

制作完成后保存
